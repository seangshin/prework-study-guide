HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • Metadata is data about our webpage that the browser uses to effectively serve our webpage to users. It is not typically seen by users, but it contains important information about accessbility, search engines, and performance.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • The most common way to style a webpage is by using an external CSS style sheet since developers keep all their CSS rules in a seperate file, which makes design changes easier.
  • The CSS box model states that every HTML element is essentially a box which consists of margins ouside the element, and the element's borders, padding, and content.
